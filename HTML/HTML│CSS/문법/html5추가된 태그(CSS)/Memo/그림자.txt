[그림자(shadow)]
[box shadow]
dx : 그림자의 가로방향 위치
dy : 그림자의 세로방향 위치
bulr : 흐려짐 정도
spread : 번짐 정도
set : 그림자 형태
ㆍoutset : 요소 바깥쪽으로 그림자가 떨어짐(기본값)
ㆍinset : 요소 안쪽으로 그림자가 떨어짐

[text shadow]
[dx] [dy] [bulr] [color]; 텍스트에 그림자를 지정

dx : 본체와 그림자의 가로방향 거리
dy : 본체와 그림자의 세로방향 거리
bulr :  그림자의 흐려짐 정도
color : 그림자의 색상

※ text-shadow는 여러가지 색상을 섞어서 그림자를 적용할 수도 있다.
  box-shadow는 한 가지의 그림자만 적용 가능.
