<input type="text" size="50" / maxlength="20">
                     ↑ 칸의 크기  ↑ 글자의 갯수


size : 글자 입력폼의 크기
      size, maxlength : 입력할 수 있는 글자의 수
      / 2가지의 속성을 지정하지 않으면 자동으로 20자 입력

      name : 텍스트 상자의 이름을 입력하며, 화면상에는 나타나지 않으나 반드시 입력해야 한다.(개발자가 읽는 부분)
      value : 텍스트 상자 안에 처음부터 내용이 보이게 하려면 value 안에 원하는 텍스트를 입력하면 된다.
